Privacy Policy

Last updated: May 20, 2026

This Privacy Policy explains how Pexels Importer handles information when used as a plugin for Adobe Premiere Pro.

Information we do not collect

Pexels Importer does not collect, sell, rent, or transmit personal data to the developer. The developer does not receive user searches, API keys, project files, downloaded media, Premiere Pro project data, or analytics from the plugin.

Pexels API key

The plugin requires each user to provide their own Pexels API key. The API key is stored locally on the user’s device using local storage. It is used only to send requests from the plugin to the Pexels API.

Third-party services

The plugin connects to Pexels services to search and download photos or videos selected by the user. Media, metadata, search results and download links are provided by Pexels and are subject to Pexels’ own terms, license and privacy practices.

Local files

When a user chooses to import media, the plugin may ask the user to select a local folder. The selected media is downloaded to that local folder and then imported into the active Premiere Pro project. The developer does not receive those files.

No backend server

The plugin does not operate a developer-controlled backend server for user searches or API keys.
