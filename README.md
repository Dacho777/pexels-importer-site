# Pexels Importer public site

This folder contains a simple GitHub Pages-ready website for Adobe Marketplace submission.

## Files
- `index.html`: public overview and usage guide
- `privacy.html`: privacy policy URL for Adobe Developer Distribution
- `terms.html`: terms of service URL for Adobe Developer Distribution
- `support.html`: support and troubleshooting page
- `assets/plugin-icon-48.png`: Adobe plugin icon 48x48
- `assets/plugin-icon-96.png`: Adobe plugin icon 96x96
- `assets/plugin-icon-192.png`: Adobe plugin icon 192x192
