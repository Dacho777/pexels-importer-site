Support

Use this page to help users install, configure and troubleshoot Pexels Importer.

Installation

  1. Install the plugin from Adobe Creative Cloud Marketplace.
  2. Open Adobe Premiere Pro.
  3. Open Window > UXP Plugins > Pexels Importer.

Getting a Pexels API key

  1. Create or sign in to your Pexels account.
  2. Go to the Pexels API page and request an API key.
  3. Copy the API key.
  4. Paste it inside the plugin and click save.

Common issues

No results appear

  • Check that your Pexels API key is valid.
  • Try a simple English search term such as “beach” or “office”.
  • Check your internet connection.

Import does not work

  • Make sure a Premiere Pro project is open.
  • Choose a valid local download folder when prompted.
  • Try importing a photo first, then test videos.

API limit reached

  • Pexels may limit API requests. Wait and try again later, or check your Pexels API account.
