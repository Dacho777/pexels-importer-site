How to use

  1. Install the plugin from Adobe Creative Cloud Marketplace.
  2. Open Adobe Premiere Pro.
  3. Go to Window > UXP Plugins > Pexels Importer.
  4. Create or obtain a free Pexels API key from Pexels.
  5. Paste the API key into the plugin and save it locally.
  6. Search for a term such as “beach”, “office”, or “city”.
  7. Select Photos or Videos.
  8. Click Import on a result to download and import the media into your active project.
